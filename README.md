This is a single page application featuring a letter-guessing game based on mathematical facts.

Game rules:
- Guess both the fact and the number
- Click letters to gradually reveal the hidden maths fact
- guess what number the fact is about

Point system:
- You start with a score of 26 points
- Maximum possible score: 30 points
- Points are deducted for every wrong letter you guess
- If you click every letter reveal button, your score turns to 0
- If you correctly guess either the fact or the number, your current score gets halved
- If you correctly guess both fact and number correct, you get +4 points
- If you get both guesses wrong, your score turns to 0
